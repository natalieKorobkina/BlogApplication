Blog Application

An Information Technology Centric Blog cite to facilitate the posting of content by non-technical users who did not have much experience with HTML or computer programming. Fitured full text search, sorting of data, integration of an RTF editor, image uploading and comments management to provide the ability of readers to leave publicly viewable comments and interact with others.
Login and registration functionality included, as well as 3-rd party  authorization 


Install

The application doesn't require any additional installation, as it published on Azure Web Pages.
